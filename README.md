# Avahi4j for Maven projects - OSGi Bundle

Simple use:

    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>

    <dependencies>
        <dependency>
            <groupId>com.github.nailtonvieira</groupId>
            <artifactId>osgi-karaf-zeroconf-avahi-bundle</artifactId>
            <version>0.1.1</version>
        </dependency>
    </dependencies>
